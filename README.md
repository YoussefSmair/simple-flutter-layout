This project is a simple Flutter UI task built using the Material framework.

It demonstrates how to structure a basic screen using common Flutter widgets such as layout containers and text elements.

The project focuses only on UI construction and does not include navigation, state management, or backend integration.

This task is intended as a practice example for understanding Flutter widget structure and layout basics.
